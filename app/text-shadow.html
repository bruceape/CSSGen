Text Shadow

Another useful CSS3 feature is Text Shadow. The name is very descriptive…it makes text have a shadow. It works very similarly to Box Shadow. Before text shadow existed, an image of text had to be used. Now we have the freedom to style text just the way we want. The syntax is simple and similar to Box Shadow.

text-shadow: 4px 2px 6px blue;
/* 4px right, 2px down, 6px blur, color blue */

Text Shadow is super easy to use, and works like Box Shadow. Also, like Box Shadow, you can stack multiple shadows with a comma.

text-shadow: 4px 2px 6px blue,
/* 4px right, 2px down, 6px blur, color blue */
             7px 0 2px green;
/* 7px right, 0px down, 2px blur, color green.

As you can see, the syntax is almost identetical to Box Shadow. Once you remember the pattern that both use, you can easily remember the pattern for other properties.

Text Shadow is safe to use in most modern browsers, and works without vendor prefixes. Text Shadow doesn't work at all in older versions of IE, so if the functionality of the site depends on Text Shadow for some reason, it's not recomended. Otherwise, go for it!
