Border Radius

Border radius is one of the most useful features of CSS3. It allows the developer to make rounded corners on boxes without images. The syntax is thankfully very easy to remember.

border-radius: 4px; /* All corners will be rounded 4px */

If you're looking to do different sized rounded corners for each corner, it's still fairly simple. Just add in where you want the corners to go.

border-top-left-radius: 4px;
border-top-right-radius: 8px;

For the shorthand property, start at the top left, and go clockwise around. This is a common pattern among CSS properties, and definitely something to memorize.

border-radius: 2px 4px 6px 5px;
/*  Top Left: 2px
    Top Right: 4px
    Bottom Right: 6px
    Bottom Left: 5px */

Border Radius is super useful, and one of the most well supported features of CSS3. It can be used safely without vendor prefixes, and will degrade gracefully. Since Border Radius generally won't affect the user experience dramatically if it isn't supported, it's a great idea to use it now.
