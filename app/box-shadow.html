Box Shadow

Box Shadow is another very useful feature of CSS3. It allows the browser to make drop shadows under elements. Box Shadow is also very well supported among the browsers. The syntax is fairly simple, and Text Shadow uses a similar pattern.

box-shadow: 4px 2px 6px 2px black;
/*  Offset 4px right, and 2px down
    Blur 6px, and spread the shadow out 2px
    Then choose a color color */

Another cool thing about Box Shadow is that multiple shadows can be applied to an element. To add more shadows, simply throw in a comma. The shadow will then stack up.

/* 4px right, 2px down, no blur or spread, black */
box-shadow: 4px 2px black,
/* 5px right, 0px down, 3px blur, no spread, orange */
            5px 0px 3px orange,
/* 1px right, 8px down, 6px blur, 2px spread, blue */
            1px 8px 6px 2px blue;

Box Shadow is widely supported among browsers, and using it is highly recommended. It does not require a vendor prefix to work in most modern browsers.
