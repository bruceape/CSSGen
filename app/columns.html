Columns

CSS3 Columns is a sweet feature that allows us to break text into multiple columns for easier reading. It's a godsend to typography on the web, and awesome that it's here for us to use. The syntax is simple. I've neglected to include the short hand, because I find that it often isn't needed.

column-count: 2;
/* Number of columns */
column-gap: 5px;
/* Space between columns */

The way to use it is pretty easy to remember. Count the columns, and specify a gap. It's that simple.

Columns have partial support in most modern browsers. Firefox and Chrome/Safari still both need vendor prefixes to work. Internet Explorer and Opera have removed the prefixes.
